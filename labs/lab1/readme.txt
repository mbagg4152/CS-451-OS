[TO COMPILE]
In the project directory, type 'make' into the terminal so that both mysort.c & myshell.c compile

[RUNNING]
In the terminal type './mysort' to run the program.

[USAGE]
In order to access the mysort program, enter 'mysort <textname>' upon seeing the prompt & hit enter, then you should see the sorted list of integers provided in the textfile.
In order to quit the program, type 'quit' into the prompt & hit enter. The program will not stop running otherwise (unless you hit ctrl+c).
